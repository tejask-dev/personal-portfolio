# Tejass Kaushik Portfolio

Premium personal brand site for Tejass Kaushik, an incoming Western Computer Science + Ivey student, AI builder, researcher, and student founder.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run check
npm run build
```

The production build outputs to `docs/`.

## Customize Content

Core profile content lives in:

```text
client/src/data/profile.ts
```

Edit this file for name, email, social links, metrics, focus areas, tech stack, and leadership items. Project and experience details currently live in their section components:

```text
client/src/components/Portfolio.tsx
client/src/components/Experience.tsx
client/src/components/Awards.tsx
```

## Design Direction

The site uses a dark navy editorial style with restrained cobalt-blue accents, subtle WebGL background motion, scroll reveals, and accessible semantic sections.
